The following parameters can be modified: model_engine: The GPT-2 model to use (default: gpt2-medium). seed: The random seed to use for reproducibility (default: 42). conversational_flows: A list of conversational flows to use (default: []). api_key: The API key to use for external services (default: null). Conversational Flows Conversational flows are predefined paths that the chatbot can take based on user inputs. They are defined in the config.json file under the conversational_flows key. Each conversational flow has a trigger_phrase and a response. The trigger_phrase is a string that the user can enter to initiate the conversational flow. The response is the chatbot's response to the trigger phrase. ~-~-~-~-~-~-~-~-~-~-~-~-~-~-~-~-~-~-~-~-~-~-~-~-~-~-~-~-~-~-~-~-~-~-~-~-~-~-~-~-~-~-~-~-~-~-~-~-~-~-~-~-~-~-~-~-~-~-~-~-~-~-~- [Acknowledgments] The Icadoa Chatbot was created by [Your Name] as a project for [Course Name].
