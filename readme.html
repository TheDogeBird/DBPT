Chatbot User Manual

Overview

The Chatbot is a conversational agent that uses natural language processing (NLP) to simulate human-like conversation. It is built on the GPT-2 (Generative Pre-trained Transformer 2) language model and uses BERT (Bidirectional Encoder Representations from Transformers) for sentiment analysis. The Chatbot can carry on a conversation on a variety of topics and can respond to user input in a conversational and engaging way.

Installation

The Chatbot requires Python 3.6 or higher and the following Python packages:

  • torch
  • transformers
  • sqlite3
  • datetime
  • json
  • random

You can install these packages using pip by running the following command:

pip install torch transformers sqlite3 datetime json random

Usage

To use the Chatbot, you can run the following command:

python main.py

This will start the Chatbot and allow you to carry on a conversation with it. You can exit the Chatbot at any time by typing 'exit'.

Customization

The Chatbot can be customized by modifying the configuration file (config.json) located in the project directory. This file contains various parameters that can be adjusted to change the behavior of the Chatbot, such as the conversational flows and the GPT-2 model configuration.
